# Infosys_Internship_2025_ClimateScope
ClimateScope —Global Weather And Air Quality Interactive Dashboard

 Overview

ClimateScope is a data analysis and visualization project focused on global weather and air quality trends.
It processes the Global Weather Repository dataset to identify temperature variations, humidity patterns, and air quality dynamics using interactive dashboards and analytical visualizations.


 Folder Structure

- data/ → Contains raw and cleaned datasets
- scripts/ → Includes project scripts ("explore_data.py", "finalised.py")
- reports/ → Holds milestone reports and documentation

---

 Project Summary

- Milestone 1: Data Cleaning & Preparation
- Milestone 2: Statistical Analysis & Visualization Design
- Milestone 3: Interactive Dashboard Development
