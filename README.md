# ClimateScope: Global Weather Analytics Dashboard

**ClimateScope** is a data-driven project that analyzes global weather patterns using the *Global Weather Repository* dataset from Kaggle. The project focuses on cleaning, analyzing, and visualizing large-scale climate data to uncover trends, correlations, and extreme weather events across regions. It includes data preprocessing, statistical exploration, and the development of an interactive visualization dashboard.

## Key Features
- Data cleaning and normalization  
- Seasonal and regional trend analysis  
- PCA-based dimensionality reduction and clustering  
- Interactive visualizations using Plotly and Streamlit  
- Insights on temperature, wind, and precipitation variations
